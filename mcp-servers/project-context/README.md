# Project Context MCP Server

Этот MCP сервер предоставляет контекст проекта logistics-platform для AI-ассистентов.

## Описание

Сервер содержит информацию о структуре проекта, архитектуре, используемых технологиях и стандартах кодирования для помощи AI в понимании контекста проекта.

## Структура проекта

Проект состоит из:
- **backend/** - NestJS backend приложение
- **frontend/** - SolidJS frontend приложение
- **supabase/** - Миграции и seed данные
- **docs/** - Документация проекта

## Технологии

- Backend: NestJS, Prisma, TypeScript
- Frontend: SolidJS, Vite, TailwindCSS
- Database: PostgreSQL (Supabase)

## Модули

Проект разделен на модули:
- drivers - управление водителями
- devices - управление устройствами
- zones - управление зонами
- addresses - управление адресами
- orders - управление заказами
- routes - управление маршрутами
- alerts - управление алертами
- activity - логирование активности

## Использование

Этот сервер предназначен для использования AI-ассистентами для лучшего понимания контекста проекта при генерации кода и решении задач.
